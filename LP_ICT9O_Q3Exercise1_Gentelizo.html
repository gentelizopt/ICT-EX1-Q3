HTML

HyperText Markup Language

HTML or Hypertext Markup Language defines the structure of web pages. It is also used to make content displayed in a web browser. HTML is also called "the most basic building block of the Web.

CSS

Cascading Style Sheets

CSS, which stands for Cascading Style Sheets is a style sheet language that has many uses. It can be used to add colors or images to the webpage, customize text colors and styles, define the cell padding of cells in a table, customize the thickness of a table's border, etc. Cascading style sheets are very useful and is incorporated in most web pages.

Bootstrap

Bootstrap Version 4

Bootstrap is a free HTML or CSS framework that helps us easily create good designs suitable for all devices. It is really easy to use and compatible with any browser. It also has responsive features and uses a mobile-first approach.
